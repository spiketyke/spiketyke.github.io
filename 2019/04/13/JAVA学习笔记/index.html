day02

1、变量就是指代在内存中开辟的存储空间，用于存放运算过程中需要用到的数据。（不同的变量类型决定了存储结构的类型）

声明三个变量int a,b,c

2、变量命名规则

  • 可以有字母、数字、“_”和“$“组成
  • 首字符不能以数字开头
  • Java大小写敏感
  • 不能使用Java保留字（一些java语言规定好的，有特殊含义的字符），如int、if、for、break等
  • 中文可以作为变量名，但不提倡用

驼峰命名，多个单词组成，第一个单词小写其余单词首字母大写

3、初始化：第一次赋值

1）声明的同时初始化：

int a=250;

2）先声明后初始化：

int a;
a=250;

c语言：可以不必初始化
int a;
输出a的值 ——– 正确（随机赋值）
java语言：
Int a;
输出a的值 ——– 错误

4、使用:

1）必需与数据类型匹配

2）对变量的操作就是对它所存的那个数的操作

3）变量在用之前必需声明并初始化

类名首字母大写

注释分三种：

1、单行注释”//“
2、多行注释”/ /“
3、文档注释”/* /“

基本数据类型

1）int:整型，4个字节，-21个多亿到21个多亿

1.1）整数直接量默认为int类型，不能超范围，否则会编译错误（直接量超范围）
1.2）两个整数相除，结果还是整数，小数位无条件舍弃
1.3）运算时超出范围会发生溢出，溢出是需要避免的（运算时超范围）

2）long：长整型，8个字节，很大很大

2.1）长整型直接量为在数字后加l或L
2.2）算数运算时有可能超范围，建议在第一个数字后加l
2.3）System.currentTimeMillis()用于获取自1970.1.1零时到此时此刻所经历的毫秒数

3）double：

3.1）浮点数直接量默认为double型表示float需要加f或F
3.2）double型数据运算时，可能会发生舍入误差

4）char：

4.1）采用Unicode编码，每个字符（char）都有一个对应的码，表现形式为char字符，实质上是int码
4.2）字符必需放在单引号中，有且仅有一个
4.3）特殊字符需要通过\来转义

5）boolean：布尔型，1个字节

5.1）只能取值为true和false

基本类型间的转换

类型从小到大依次为：
byte,short,int,long,float,double
           ↑
        char

1）两种方式：

1.1）自动类型转换：从小类型到大类型
1.2）强制类型转换：从大类型到小类型
强转有可能溢出或精度丢失

2）两点规则：

2.1）整数直接量可以直接赋值给byte、short、char但不能超范围
2.2）byte、short、char型变量参与运算时，先一律转为int再运算
1B=8Bit（位）

Scanner的用法：

1）在package下：

import java.util.Scanner;

给变量赋值三种方式：

  3）系统随机生成一个值

2、分支结构：

1）if结构：1条路
2）if… else结构：2条路

day04

1、循环：反复执行一段相同或相似的代码

2、循环三要素：

1）循环变量的初始化
2）循环的条件（以循环变量为基础）
3）循环变量的改变（向着循环的结束变）
循环变量：在循环过程中改变的那个量

3、循环结构

1）while：先判断后执行，有可能一次都不执行
2）do…while结构：先执行一次后判断，至少执行一次
第一要素与第三要素相同时首选
3）for结构：应用率最高

continue：跳过循环体剩余语句而进入下一次循环

1）顺序：从上往下逐行执行，每句必走
2）分支：有条件的执行某语句一次，并非每句必走
3）循环：有条件的执行某语句多次，并非每句必走

day05

1、三种循环结构的更加实用情况：

1）while：“当…”循环
2）do…while：“直到…”循环
第1要素与第3要素相同时首选
3）for：固定次数循环，应用率最高

2、嵌套循环：

1）循环中套循环，一般多行多列时使用，外层控制行，内层控制列
2）执行过程：外层循环走一次，内层循环走所有次
3）建议：嵌套层次越少越好（一般最多两层循环）
若逻辑必需通过3层以上的循环才能解决，说明设计有问题
4）break只能跳出一层循环（想一下子跳出所有层循环，这样的需求没 有）

冒泡的工作原理：
8.1）4个数冒三轮
8.2）每一轮都是从第一个元素开始冒
每一次都是和它的下一个元素比

找最大值的算法：
1、假设第一个元素为最大值
int max=arr[0];
2、遍历剩余元素，将剩余元素与max对比，若剩余元素大于max，修改max为较大的数
for(int i=0;i<arr.length;i++){
if(arr[i]>max){
max=arr[i];
}
}
